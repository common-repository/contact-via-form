A Simple contact form plugin

== Description ==

This is simple contact form plugin that sends email to the user.

This contact form plugin provides an interface for sending email.

<h3>All Features:</h3>
<ul>
	<li>Use this shortcode [cvf_form] to display the form and accept submissions from frontend.</li>
	<li><strong>Contact via form Plugin</strong> will create a form where your visitors can submit their posts.</li>
	<li>Simple and user friendly</li>
</ul>


== Installation ==

You will be able to add so simple and user friendly contacting feature for your visitors within a short moment.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

<strong>Or</strong>

1. Upload `Contact via form.zip` to the `/wp-content/plugins/` directory.

2. Activate the plugin through the 'Plugins' menu in WordPress Dashboard.
